# Programming

Architectural programming skills for Claude Code — occupancy load calculations and workplace space programming.

## Skills

| Skill | Description |
|-------|-------------|
| [occupancy-calculator](skills/occupancy-calculator/) | IBC occupancy load calculator — per-area occupant loads, gross vs net area, egress requirements |
| [workplace-programmer](skills/workplace-programmer/) | Workplace strategy consultant — area splits, room schedules, seat counts backed by industry research |

## Commands

| Command | Description |
|---------|-------------|
| [space-program](commands/space-program.md) | Build a complete space program — occupancy calculation followed by workplace programming |

## Install

```bash
git clone https://github.com/AlpacaLabsLLC/skills.git
# Then symlink individual skills or install the full plugin
```
