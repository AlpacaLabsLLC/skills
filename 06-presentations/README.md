# Presentations

Presentation skills for Claude Code — editorial slide decks and color palette generation.

## Skills

| Skill | Description |
|-------|-------------|
| [slide-deck-generator](skills/slide-deck-generator/) | Generates self-contained HTML slide decks with 22 slide types and an editorial design system |
| [color-palette-generator](skills/color-palette-generator/) | Creates harmonious color palettes from descriptions, moods, images, or brand references with WCAG contrast checks |

## Install

```bash
claude install github:AlpacaLabsLLC/skills-for-architects/06-presentations
```
