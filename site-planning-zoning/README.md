# Site Planning & Zoning

Site planning and zoning skills for Claude Code — site research, zoning envelope analysis, and design brief generation.

## Skills

| Skill | Description |
|-------|-------------|
| [site-analysis-generator](skills/site-analysis-generator/) | Researches a site and produces a comprehensive analysis brief (climate, zoning, transit, demographics, environment) |
| [zoning-analyzer](skills/zoning-analyzer/) | Zoning envelope analyzer for Maldonado, Uruguay — parses GIS data and calculates height, setbacks, FOS/FOT from TONE regulations |
| [design-brief-builder](skills/design-brief-builder/) | Turns vague client requirements into structured design briefs ready for schematic design |

## Commands

| Command | Description |
|---------|-------------|
| [site-due-diligence](commands/site-due-diligence.md) | Full site due diligence — site analysis followed by zoning envelope calculation |

## Install

```bash
git clone https://github.com/AlpacaLabsLLC/skills.git
# Then symlink individual skills or install the full plugin
```
